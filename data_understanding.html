Data Understanding#

Data Understanding (Pemahaman Data)#

Data Understanding adalah tahap pertama dalam proses Data Science atau Data Analytics yang berfokus pada pengumpulan dan pemahaman data yang tersedia. Tujuan dari tahap ini adalah untuk mengeksplorasi dan memahami struktur, kualitas, dan karakteristik data yang akan digunakan dalam analisis. Proses ini melibatkan beberapa aktivitas utama, seperti:

  1. Pengumpulan Data: Mengumpulkan data dari berbagai sumber yang relevan untuk analisis.

  2. Eksplorasi Data: Menganalisis data secara visual dan statistik untuk memahami pola, tren, dan hubungan antar variabel.

  3. Pembersihan Data: Mengidentifikasi dan menangani masalah data seperti missing values, duplikasi, atau kesalahan dalam data.

  4. Identifikasi Variabel Kunci: Menentukan fitur atau variabel yang memiliki pengaruh besar terhadap tujuan analisis.

  5. Pahami Distribusi Data: Memeriksa distribusi nilai-nilai variabel untuk memahami sebaran data.

Dengan data yang dipahami dengan baik, analisis berikutnya dapat dilakukan lebih efektif dan menghasilkan insight yang lebih akurat.

Langkah-Langkah Dalam Data Understanding#

Berikut adalah langkah-langkah dalam Data Understanding:

  1. Pengumpulan Data:

    • Mengumpulkan data dari berbagai sumber (misalnya, database, file CSV, API, dll.).

    • Memastikan data yang dikumpulkan relevan dengan tujuan analisis.

  2. Eksplorasi Data:

    • Menggunakan teknik statistik dan visualisasi untuk melihat pola atau hubungan antar variabel.

    • Membuat statistik deskriptif (mean, median, modus, standar deviasi) untuk memahami distribusi data.

    • Menggunakan grafik seperti histogram, boxplot, scatter plot untuk mengeksplorasi hubungan antar data.

  3. Pembersihan Data (Data Cleaning):

    • Mengidentifikasi dan menangani nilai yang hilang (missing values) dengan mengganti, menghapus, atau mengimputasi data.

    • Mengidentifikasi dan menghapus duplikasi data jika ada.

    • Menangani data yang tidak konsisten (misalnya, kesalahan penulisan atau format data yang salah).

  4. Identifikasi Outliers:

    • Mencari nilai ekstrim atau outliers yang bisa mempengaruhi analisis.

    • Menggunakan teknik statistik atau visualisasi (seperti boxplot) untuk mendeteksi outliers.

    • Menentukan apakah outliers perlu dihapus, diperbaiki, atau dibiarkan.

  5. Memahami Struktur Data:

    • Memeriksa jenis data (numerik, kategorikal, teks) dan hubungan antar variabel.

    • Menyusun data dalam bentuk yang sesuai untuk analisis lebih lanjut (misalnya, mengonversi data kategorikal menjadi format numerik jika diperlukan).

  6. Pemeriksaan Korelasi Antar Variabel:

    • Menggunakan teknik statistik seperti korelasi Pearson atau Spearman untuk melihat hubungan antara variabel.

    • Menentukan variabel mana yang saling mempengaruhi dan mana yang tidak.

  7. Validasi Sumber Data:

    • Memastikan bahwa data yang digunakan memiliki kualitas yang baik dan dapat dipercaya.

    • Memeriksa apakah ada bias dalam data yang bisa mempengaruhi hasil analisis.

  8. Dokumentasi Temuan:

    • Mencatat semua temuan selama eksplorasi data, seperti pola yang ditemukan, masalah yang muncul, dan langkah-langkah pembersihan yang telah dilakukan.

    • Menggunakan dokumentasi ini untuk merencanakan langkah-langkah analisis lebih lanjut.

Setelah langkah-langkah ini selesai, data akan siap untuk diproses lebih lanjut dalam tahap analisis dan modeling.

Jenis Tipe di Data Mining#

Ketika kita mempelajari data mining tentunya objek yang paling utama diolah adalah data itu sendiri. Data adalah sekumpulan fakta atau objek dan atributnya. Atribut merupakn karakteristik atau properti dari objek tertentu. Jenis atribut atau nilai dari kolom ada beberapa, diantaranya adalah :

  • Nominal yang merupakan kategori class yang tidak memiliki perbedaan nilai atau rangking. Contoh pekerjaan (Dosen, PNS).

  • Binary merupakan atribut yang tidak memiliki perbedaan dan hanya memiliki 2 states, contohnya seperti jenis kelamin (perempuan, laki-laki).

  • Ordinal merupakan kategori class yang memiliki nilai rangking, contohnya seperti jabatan, jenjang pendidikan, dll.

  • Numeric merupakan kuantitas yang memiliki ukuran nilai yang memiliki jarak yang jelas, contohnya adalah umur. Numeric ini memiliki 2 jenis, yaitu rasio dan interval.

DBeaver.#

DBeaver adalah alat manajemen basis data (database management tool) yang bersifat open-source dan mendukung berbagai jenis database, seperti MySQL, PostgreSQL, SQLite, Oracle, SQL Server, dan lainnya. DBeaver menyediakan antarmuka grafis (GUI) yang memudahkan pengguna untuk melakukan query, eksplorasi data, manajemen skema, dan administrasi database.

Fitur Utama DBeaver#

Dukungan Multi-Database – Bisa terhubung ke berbagai sistem database.
Query Editor – Mendukung SQL editor dengan fitur auto-complete.
ER Diagram – Bisa menampilkan struktur database dalam bentuk diagram.
Data Import/Export – Mendukung ekspor data ke format seperti CSV, JSON, XML, Excel.
Open-Source & Cross-Platform – Bisa digunakan di Windows, macOS, dan Linux.

DBeaver sering digunakan oleh developer, database administrator (DBA), dan data analyst untuk mengelola database dengan lebih efisien.

Berikut langkah-langkah untuk menghubungkan DBeaver ke database:#

Berikut langkah-langkah untuk menghubungkan DBeaver ke database:

1. Install DBeaver#

Jika belum memiliki DBeaver, unduh dan instal dari https://dbeaver.io/download/.
Gunakan DBeaver Community (gratis) atau DBeaver Enterprise (berbayar jika butuh fitur tambahan).

2. Buka DBeaver dan Tambahkan Koneksi Database#

  1. Buka DBeaver setelah instalasi selesai.

  2. Klik File → New → Database Connection atau tekan tombol Ctrl + N.

  3. Pilih jenis database yang ingin dikoneksikan, misalnya MySQL, PostgreSQL, atau lainnya.

  4. Klik Next.

3. Masukkan Detail Koneksi#

  1. Host: Masukkan alamat server database (contoh: localhost atau IP server).

  2. Port: Gunakan port default atau sesuai konfigurasi (contoh:

    • MySQL: 3306

    • PostgreSQL: 5432

  3. Database Name: Masukkan nama database yang ingin dihubungkan.

  4. Username & Password: Masukkan kredensial pengguna database.

  5. Klik “Test Connection” untuk memastikan koneksi berhasil.

  6. Jika diminta, instal driver database dengan menekan tombol Download Driver.

4. Selesai & Mulai Menggunakan DBeaver#

  • Jika koneksi berhasil, klik Finish.

  • Database akan muncul di panel Database Navigator di sisi kiri.

  • Sekarang Anda bisa menjalankan SQL Query, melihat tabel, mengedit data, dan mengelola database.

Jika Anda menggunakan database di cloud seperti Aiven.io, pastikan:
Anda memiliki hostname, port, username, dan password dari Aiven.
Pastikan server database mengizinkan koneksi dari IP Anda.

Algoritma K-Nearest Neighbor (KNN)#

Algoritma KNN atau K-Nearest Neighbor adalah salah satu algoritma yang banyak digunakan di dunia machine learning untuk kasus klasifikasi.

Apa itu Algoritma KNN?#

Seperti yang telah disebutkan, algoritma KNN merupakan algoritma klasifikasi yang bekerja dengan mengambil sejumlah K data terdekat (tetangganya) sebagai acuan untuk menentukan kelas dari data baru. Algoritma ini mengklasifikasikan data berdasarkan similarity atau kemiripan atau kedekatannya terhadap data lainnya.

Dalam K-Nearest Neighbor, data point yang berada berdekatan disebut “neighbor” atau “tetangga”.

Secara umum, cara kerja algoritma KNN adalah sebagai berikut.

  1. Tentukan jumlah tetangga (K) yang akan digunakan untuk pertimbangan penentuan kelas.

  2. Hitung jarak dari data baru ke masing-masing data point di dataset.

  3. Ambil sejumlah K data dengan jarak terdekat, kemudian tentukan kelas dari data baru tersebut.

Menghitung jarak dengan Euclidean Distance#

Untuk menghitung jarak antara dua titik pada algoritma KNN digunakan metode Euclidean Distance yang dapat digunakan pada 1-dimensional space, 2-dimensional space, atau multi-dimensional space.

1-dimensional space berarti perhitungan jarak hanya menggunakan satu variabel bebas (independent variable), 2-dimensional-space berarti ada dua variabel bebas, dan multi-dimensional space berarti ada lebih dari dua variabel.

Secara umum, formula Euclidean distance pada 1-dimensional space adalah sebagai berikut.

Formula di atas dapat digunakan jika jumlah independent variable hanya ada satu variabel. Lalu, bagaimana jika ada banyak variabel yang digunakan?

Jika ada lebih dari satu, kita dapat menjumlahkannya seperti di bawah ini.

# 📌 Import library yang diperlukan
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.neighbors import NearestNeighbors  # Model KNN untuk mendeteksi outlier
from IPython.display import display  # Impor display untuk digunakan di Google Colab

# Pilih hanya kolom numerik dari data yang telah digabungkan (df_combined)
numerical_columns = df_combined.select_dtypes(include=[np.number]).columns

# Periksa apakah ada kolom numerik dalam dataset
if len(numerical_columns) == 0:
    print("⚠ Tidak ada kolom numerik untuk dianalisis.")
else:
    # Ambil hanya nilai numerik dari dataset untuk analisis outlier
    X = df_combined[numerical_columns].values

    # Tangani nilai NaN dengan menggantinya menggunakan nilai median dari setiap kolom
    X = np.nan_to_num(X, nan=np.nanmedian(X))

    # Tentukan jumlah tetangga (k) berdasarkan aturan umum: k = akar jumlah data
    k = max(3, int(np.sqrt(len(X))))  # Minimal k=3 untuk memastikan stabilitas perhitungan

    # Latih model KNN dan cari k tetangga terdekat untuk setiap data
    nbrs = NearestNeighbors(n_neighbors=k+1).fit(X)
    distances, indices = nbrs.kneighbors(X)

    # Ambil rata-rata jarak dari k tetangga terdekat (mengabaikan titik data itu sendiri)
    avg_distances = distances[:, 1:].mean(axis=1)

    # Tentukan threshold untuk mendeteksi outlier menggunakan persentil 99.5
    threshold = np.percentile(avg_distances, 99.5)

    # Temukan indeks data yang memiliki rata-rata jarak lebih besar dari threshold
    outliers = np.where(avg_distances > threshold)[0]

    # Tampilkan hasil jumlah outlier yang ditemukan
    print(f"📌 Total outliers ditemukan: {len(outliers)}\n")

    # Jika ada outlier, tampilkan indeks dan data outlier
    if len(outliers) > 0:
        print("Indeks Outliers:", outliers)
        print("\n🔍 Data Outliers:")
        display(df_combined.iloc[outliers])  # Gunakan display() dari IPython untuk Colab

        # Menghitung dan menampilkan rank berdasarkan jarak rata-rata
        outlier_distances = avg_distances[outliers]
        sorted_indices = np.argsort(outlier_distances)[::-1]  # Urutkan jarak outlier dari yang terbesar
        print("\n🔍 Rank Outliers berdasarkan jarak rata-rata:")
        for rank, idx in enumerate(sorted_indices, start=1):
            print(f"Rank {rank}: Indeks {outliers[idx]}, Jarak Rata-rata: {outlier_distances[idx]}")

    # Visualisasi outlier hanya jika data memiliki 2 fitur numerik (agar bisa digambarkan dalam 2D)
    if X.shape[1] == 2:
        plt.figure(figsize=(8,6))
        plt.scatter(X[:, 0], X[:, 1], label="Normal Data", alpha=0.6)  # Data normal
        plt.scatter(X[outliers, 0], X[outliers, 1], color='red', label="Outlier", edgecolors='k', s=100)  # Data outlier
        plt.xlabel(numerical_columns[0])
        plt.ylabel(numerical_columns[1])
        plt.title("Outlier Detection using KNN")
        plt.legend()
        plt.show()
