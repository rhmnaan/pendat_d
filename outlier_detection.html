Outlier detection with Local Outlier Factor (LOF)#

Faktor outlier lokal (LOF)#

Apa itu Faktor outlier lokal (LOF)?#

Faktor outlier lokal (LOF) adalah algoritma yang mengidentifikasi outlier yang ada dalam dataset.

apa yang dimaksud dengan outlier lokal ?#

Ketika suatu titik dianggap sebagai outlier berdasarkan lingkungan lokalnya, itu adalah outlier lokal . LOF akan mengidentifikasi outlier dengan mempertimbangkan kepadatan lingkungan. LOF berkinerja baik ketika kepadatan data tidak sama di seluruh kumpulan data.

Untuk memahami LOF, kita harus mempelajari beberapa konsep secara berurutan:

K-jarak dan K-tetangga

Jarak jangkauan (RD)

Kepadatan jangkauan lokal (LRD)

Faktor Pencilan Lokal (LOF)

K-distance adalah jarak antara titik, dan itu adalah tetangga terdekat Kᵗʰ.

K-tetangga dilambangkan dengan Nₖ(A) termasuk himpunan titik-titik yang terletak pada atau pada lingkaran dengan radius K-jarak. K-neighbors bisa lebih dari atau sama dengan nilai K

KEPADATAN REACHABILITY (RD)#

Ini didefinisikan sebagai jarak K maksimum dari Xj dan jarak antara Xi dan Xj. Ukuran jarak adalah masalah khusus (Euclidean, Manhattan, dll.)

KEPADATAN JANGKAUAN LOKAL (LRD)#

LRD adalah kebalikan dari jarak jangkauan rata-rata A dari tetangganya. Secara intuitif menurut rumus LRD, semakin banyak jarak jangkauan rata-rata (yaitu, tetangga jauh dari titik), semakin sedikit kepadatan titik yang ada di sekitar titik tertentu. Ini menunjukkan seberapa jauh suatu titik dari kumpulan titik terdekat. Nilai LRD yang rendah menyiratkan bahwa cluster terdekat jauh dari titik.

FAKTOR OUTLIER LOKAL (LOF)#

LRD setiap titik digunakan untuk membandingkan dengan rata-rata LRD dari K tetangganya. LOF adalah rasio rata-rata LRD dari K tetangga A terhadap LRD dari A.

Secara intuitif, jika titik tersebut bukan merupakan outlier (inlier), maka rasio rata-rata LRD tetangga kira-kira sama dengan LRD suatu titik (karena densitas suatu titik dan tetangganya kira-kira sama). Dalam hal ini, LOF hampir sama dengan 1. Sebaliknya, jika titik tersebut merupakan outlier, LRD suatu titik lebih kecil dari rata-rata LRD tetangganya. Maka nilai LOF akan tinggi.

Umumnya jika LOF > 1 dianggap sebagai outlier, namun hal tersebut tidak selalu benar. Katakanlah kita tahu bahwa kita hanya memiliki satu outlier dalam data, kemudian kita mengambil nilai LOF maksimum di antara semua nilai LOF, dan titik yang sesuai dengan nilai LOF maksimum akan dianggap sebagai outlier.

KEUNGGULAN LOF#

Sebuah titik akan dianggap sebagai outlier jika berada pada jarak kecil ke cluster yang sangat padat. Pendekatan global mungkin tidak mempertimbangkan hal itu sebagai sebuah outlier. Tetapi LOF dapat secara efektif mengidentifikasi outlier lokal.

KEKURANGAN LOF#

Karena LOF adalah rasio, sulit untuk ditafsirkan. Tidak ada nilai ambang batas tertentu di mana suatu titik didefinisikan sebagai outlier. Identifikasi outlier tergantung pada masalah dan pengguna.

KESIMPULAN#

Nilai faktor outlier lokal (LOF) mengidentifikasi outlier berdasarkan lingkungan lokal. Ini memberikan hasil yang lebih baik daripada pendekatan global untuk menemukan outlier. = (K36 / K34 + K35 / K34 + K38 / K34) / 3

# 3. Menghitung Reachability Distance (RD)
def reachability_distance(k_distance, distance_matrix, final_indices):
    rd_matrix = np.zeros_like(distance_matrix)
    for i in range(len(distance_matrix)):
        for j_idx, j in enumerate(final_indices[i]):  # Hanya tetangga yang dipertimbangkan
            rd_matrix[i, j] = max(k_distance[i], distance_matrix[i, j])  # Menggunakan k-Distance(A)
    return rd_matrix

rd_matrix = reachability_distance(k_distance, distance_matrix, final_indices)

# Konversi RD ke DataFrame
df_rd = pd.DataFrame(rd_matrix, index=labels, columns=labels)
print("\n=== Reachability Distance (RD) ===")
print(df_rd.round(4))

# 4. Menghitung Local Reachability Density (LRD)
def local_reachability_density(final_indices, rd_matrix):
    lrd = np.zeros(len(final_indices))
    for i in range(len(final_indices)):
        reach_dists = [rd_matrix[i, j] for j in final_indices[i]]
        lrd[i] = k / sum(reach_dists) if sum(reach_dists) > 0 else 0  # Hindari pembagian dengan nol
    return lrd

lrd_values = local_reachability_density(final_indices, rd_matrix)

# Konversi LRD ke DataFrame
df_lrd = pd.DataFrame({"ID": labels, "LRD": lrd_values.round(4)})

# 5. Menghitung Local Outlier Factor (LOF)
def local_outlier_factor(final_indices, lrd_values):
    lof = np.zeros(len(final_indices))
    for i in range(len(final_indices)):
        neighbor_lrd_sum = sum(lrd_values[j] for j in final_indices[i])
        lof[i] = (neighbor_lrd_sum / (k * lrd_values[i])) if lrd_values[i] > 0 else 0  # Hindari pembagian dengan nol
    return lof

lof_values = local_outlier_factor(final_indices, lrd_values)

# Konversi LOF ke DataFrame
df_lof = pd.DataFrame({"ID": labels, "LOF": lof_values.round(4)})
print("\n=== Local Outlier Factor (LOF) ===")
print(df_lof)

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.legend_handler import HandlerPathCollection
from sklearn.neighbors import LocalOutlierFactor

# Data titik (X, Y)
data = np.array([
    [2.00, 3.00], [2.50, 2.80], [1.80, 3.20], [8.00, 8.50], [7.50, 7.80],
    [8.20, 8.30], [50.00, 50.00], [55.00, 53.00], [10.00, -12.00], [-12.00, -10.50]
])

# Menerapkan LOF untuk deteksi outlier
lof = LocalOutlierFactor(n_neighbors=5)
X_scores = -lof.fit_predict(data)  # Skor outlier
n_errors = (X_scores == -1).sum()

# Fungsi untuk menyesuaikan ukuran marker di legenda
def update_legend_marker_size(handle, orig):
    "Customize size of the legend marker"
    handle.update_from(orig)
    handle.set_sizes([20])

# Plot scatter plot
plt.figure(figsize=(8, 6))
radius = (X_scores.max() - X_scores) / (X_scores.max() - X_scores.min())
scatter = plt.scatter(
    data[:, 0],
    data[:, 1],
    s=1000 * radius,
    edgecolors="r",
    facecolors="none",
    label="Outlier scores",
)
plt.scatter(data[:, 0], data[:, 1], color="k", s=3.0, label="Data points")
plt.axis("tight")
plt.xlabel("Prediction errors: %d" % (n_errors))
plt.legend(
    handler_map={scatter: HandlerPathCollection(update_func=update_legend_marker_size)}
)
plt.title("Local Outlier Factor (LOF)")
plt.show()
