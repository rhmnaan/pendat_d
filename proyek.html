Pemodelan (Model Klasifikasi)#

  1. Bagi data menjadi data latih dan data uji

  2. Bangun model klasifikasi (di sini kita pakai Random Forest karena cocok untuk data tabular)

  3. Latih model

from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# 1. Split data menjadi train dan test
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 2. Inisialisasi model Random Forest
model = RandomForestClassifier(random_state=42)

# 3. Latih model dengan data latih
model.fit(X_train, y_train)

# 4. Cek akurasi pada data latih
train_score = model.score(X_train, y_train)
print(f"‚úÖ Model berhasil dilatih. Akurasi di data latih: {train_score:.2f}")

‚úÖ Model berhasil dilatih. Akurasi di data latih: 1.00

Evaluasi Model#

  1. Prediksi data uji

  2. Hitung metrik evaluasi: akurasi, precision, recall, f1-score

  3. Tampilkan confusion matrix dan classification report

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score, confusion_matrix, classification_report
import seaborn as sns
import matplotlib.pyplot as plt

# 1. Prediksi data uji
y_pred = model.predict(X_test)

# 2. Evaluasi metrik
print("üîç Evaluasi Model:")
print(f"üîπ Akurasi       : {accuracy_score(y_test, y_pred):.2f}")
print(f"üîπ Precision     : {precision_score(y_test, y_pred, average='weighted'):.2f}")
print(f"üîπ Recall        : {recall_score(y_test, y_pred, average='weighted'):.2f}")
print(f"üîπ F1 Score      : {f1_score(y_test, y_pred, average='weighted'):.2f}")

# 3. Confusion matrix
cm = confusion_matrix(y_test, y_pred)
plt.figure(figsize=(6,4))
sns.heatmap(cm, annot=True, fmt='d', cmap='Blues')
plt.title("Confusion Matrix")
plt.xlabel("Predicted")
plt.ylabel("Actual")
plt.show()

# 4. Classification report
print("\nüìã Classification Report:")
print(classification_report(y_test, y_pred))
