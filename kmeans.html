K Means#

Langkah-langkah Algoritma K Means#

  1. Tentukan Jumlah Cluster (K) Contoh : Dibagi menjadi 2 Cluster

  2. Pilih K Centroid Awal Secara Acak Dari Data Berapa Saja.

  3. Hitung Jarak Setiap data ke tiap Centroid

  4. Kelompokkan data berdasarkan centroid terdekat

  5. Hitung ulang centroid dari rata-rata anggota cluster

  6. Ulangi langkah 3-5 hingga konvergen( tidak ada perubahan signifikan)

Rumus-rumus yang digunakan#

Euclidean distance

Inertia ( Sum Square Error / SSE)

Silhouette Score

import pandas as pd
import numpy as np
from sklearn.datasets import load_iris
from scipy.spatial.distance import cdist

# Memuat dataset Iris
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)

# Mengganti nama kolom sesuai keinginan
df.columns = ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']

# Inisialisasi centroid awal secara manual (2 cluster)
np.random.seed(0)
centroid_indices = np.random.choice(df.index, 2, replace=False)
centroids = df.iloc[centroid_indices].values

# Menampilkan indeks dan nilai centroid awal
print("\nðŸŸ¡ Indeks Centroid Awal (Iterasi ke-0):")
for i, index in enumerate(centroid_indices):
    print(f"Centroid {i}: Data ke-{index} -> {centroids[i]}")

# Menghitung jarak Euclidean ke setiap centroid
distances = cdist(df.values, centroids, metric='euclidean')

# Menentukan jarak terdekat dan cluster yang diikuti
df['Distance_to_Centroid_0'] = distances[:, 0]
df['Distance_to_Centroid_1'] = distances[:, 1]
df['Jarak_Terdekat'] = df[['Distance_to_Centroid_0', 'Distance_to_Centroid_1']].min(axis=1)
df['Cluster_Diikuti'] = np.where(df['Distance_to_Centroid_0'] <= df['Distance_to_Centroid_1'], 0, 1)

# Menampilkan hasil iterasi pertama dengan semua kolom, dengan kolom yang diperbarui
print("\nðŸ“ Hasil Iterasi ke-1:")
print(df[['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm',
          'Distance_to_Centroid_0', 'Distance_to_Centroid_1', 'Jarak_Terdekat', 'Cluster_Diikuti']].head(50).to_string(index=True))

Clustering Dataset Iris dengan 2 Cluster Menggunakan K-Means#

import pandas as pd
import numpy as np
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans

# Memuat dataset Iris
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)

# Mengganti nama kolom sesuai keinginan
df.columns = ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']

# Menambahkan kolom 'Id' untuk simulasi ID
df['Id'] = df.index + 1  # Menambahkan ID yang dimulai dari 1

# Ambil centroid awal berdasarkan ID tertentu
centroid_0 = df.loc[df['Id'] == 1, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0]
centroid_1 = df.loc[df['Id'] == 114, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0]

# Gabungkan ke dalam array untuk inisialisasi centroid awal
initial_centroids = np.array([centroid_0, centroid_1])

# Inisialisasi KMeans dengan k=2
kmeans = KMeans(n_clusters=2, init=initial_centroids, n_init=1, max_iter=100, random_state=42)

# Fit model ke data
kmeans.fit(df[['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']])

# Simpan label cluster ke dataframe
df['Cluster'] = kmeans.labels_

# Menampilkan centroid akhir setelah clustering

Evaluasi Cluster untuk Dataset Iris dengan Inisialisasi Centroid Manual pada Metode K-Means#

Deskripsi Singkat: Code ini melakukan clustering pada dataset Iris menggunakan metode K-Means dengan inisialisasi centroid manual untuk 2 hingga 4 cluster. Evaluasi dilakukan menggunakan Inertia dan Silhouette Score untuk menilai kualitas pembagian cluster.

import pandas as pd
import numpy as np
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

# Memuat dataset Iris
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)

# Mengganti nama kolom sesuai keinginan
df.columns = ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']

# Menambahkan kolom 'Id' untuk simulasi ID
df['Id'] = df.index + 1  # Menambahkan ID yang dimulai dari 1

# Menyimpan fitur untuk clustering
features = df[['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values

# Menyiapkan tabel untuk menyimpan hasil
results = []

# Loop untuk 2 hingga 4 cluster
for n_clusters in range(2, 5):
    # Ambil centroid awal berdasarkan ID tertentu untuk 3 dan 4 cluster
    if n_clusters == 2:
        initial_centroids = np.array([
            df.loc[df['Id'] == 1, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 114, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0]
        ])
    elif n_clusters == 3:
        initial_centroids = np.array([
            df.loc[df['Id'] == 10, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 100, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 50, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0]
        ])
    elif n_clusters == 4:
        initial_centroids = np.array([
            df.loc[df['Id'] == 1, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 114, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 50, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0],
            df.loc[df['Id'] == 90, ['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']].values[0]
        ])

    # Inisialisasi dan fit KMeans
    kmeans = KMeans(n_clusters=n_clusters, init=initial_centroids, n_init=1, max_iter=100, random_state=42)
    kmeans.fit(features)

    # Menghitung inertia dan silhouette score
    inertia = kmeans.inertia_
    sil_score = silhouette_score(features, kmeans.labels_)

    # Menyimpan hasil ke dalam list
    results.append([n_clusters, inertia, sil_score, kmeans.n_iter_])

    # Tampilkan hasil cluster
    print(f"\nðŸ”¹ **{n_clusters} Cluster**")
    print("Centroid Akhir:")
    print(pd.DataFrame(kmeans.cluster_centers_, columns=['SepalLengthCm', 'SepalWidthCm', 'PetalLengthCm', 'PetalWidthCm']))
    print(f"\nInertia: {inertia:.4f}")
    print(f"Silhouette Score: {sil_score:.4f}")
    print(f"Jumlah Iterasi sampai konvergen: {kmeans.n_iter_}")

# Konversi hasil ke DataFrame untuk tampilan yang lebih baik
results_df = pd.DataFrame(results, columns=['k', 'Inertia', 'Silhouette Score', 'Iterasi'])
print("\nðŸ” **Evaluasi untuk Berbagai Nilai k:**")
print(results_df.to_string(index=False))
